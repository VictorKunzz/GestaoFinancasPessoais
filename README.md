# Gestão Finanças Pessoais

Projeto desenvolvido para a disciplina de Programação Web no curso de Engenharia de Software.

A ideia é criar um sistema simples de controle financeiro pessoal, mas com alguns diferenciais que normalmente não existem em dashboards básicos.

---

##  Objetivo

Ajudar o usuário a controlar sua vida financeira de forma visual, simples e até um pouco inteligente, mostrando não só os gastos, mas também **insights e previsões**.

---

##  Funcionalidades principais

- Cadastro e login de usuário
- Registro de receitas e despesas
- Dashboard com saldo total
- Gráficos de gastos por categoria
- Filtro por período

---

##  Funcionalidades Diferenciadas

O sistema também conta com ideias diferentes do usual:

###  Score de saúde financeira
Um número de 0 a 100 que mostra o quão saudável estão suas finanças.

---

###  Insights automáticos
O sistema analisa seus dados e mostra coisas como:

- “Você gastou mais do que o normal esse mês”
- “Seu maior gasto foi com alimentação”
- “Se continuar assim, seu saldo acaba em X dias”

---

###  Previsão de saldo
Mostra uma projeção do seu saldo futuro baseado no seu padrão atual de gastos.

---

###  Metas com previsão de conclusão
O sistema calcula em quanto tempo você consegue atingir uma meta.

---

###  Classificação automática de gastos
Se o usuário digitar algo tipo “Uber” ou “McDonalds”, o sistema tenta classificar sozinho.

---

###  Gamificação
Pequenos incentivos como:
- medalhas
- metas atingidas
- evolução de mês a mês
